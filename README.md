# PRODIGY_DS_02

## Data Cleaning and Exploratory Data Analysis (EDA)

### Objective

Perform data cleaning and exploratory data analysis on a dataset to identify patterns, trends, relationships, and key business insights.

### Dataset

**Sample Superstore Dataset**

The dataset contains information about:

* Sales
* Profit
* Product Categories
* Regions
* Customers
* Orders

### Tools Used

* Python
* Pandas
* Matplotlib
* Visual Studio Code

### Tasks Performed

* Loaded and explored the dataset
* Checked dataset structure and missing values
* Analyzed sales performance across product categories
* Analyzed profit distribution across categories
* Examined sales distribution by region
* Visualized sales distribution using a histogram

### Visualizations Created

1. Sales by Category
2. Profit by Category
3. Sales Distribution by Region
4. Sales Histogram

### Key Insights

* Different product categories contribute differently to overall sales.
* Profitability varies across categories.
* Sales are distributed unevenly across regions.
* Most sales values are concentrated within a specific range, with some high-value transactions acting as outliers.
